# Customer Behavior Analysis using SQL, Python & Power BI

## Overview

This project is an end-to-end customer behavior analysis that combines SQL, Python, and Power BI to transform raw customer shopping data into meaningful business insights.

The project demonstrates a complete analytics workflow, from data cleaning and exploration to visualization and dashboard creation, helping identify customer purchasing patterns and support data-driven business decisions.

---

## Business Problem

Retail businesses generate large volumes of customer transaction data, but extracting actionable insights can be challenging.

The objective of this project is to analyze customer shopping behavior to answer questions such as:

- Who are the most valuable customers?
- Which products perform best?
- What purchasing trends exist?
- Which customer segments generate the highest revenue?
- How can businesses improve customer engagement and sales?

---

## Dataset

The dataset contains customer shopping transactions, including information such as:

- Customer ID
- Age
- Gender
- Product Category
- Quantity
- Price

- Payment Method
- Shopping Mall
- Total Sales
